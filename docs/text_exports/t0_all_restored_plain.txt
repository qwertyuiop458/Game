нажмите 5, чтобы использовать оружие (у взаимодействия есть приоритет).
нажмите в направлении движения объекта, чтобы схватить его. удерживайте, чтобы толкнуть его. схватив, нажмите 5, чтобы далеко его забросить.
нажмите 0, чтобы перезарядить оружие (или 5/OK, когда оружие разрядилось).
нажмите *, чтобы быстро взять другое оружие. нажмите #, чтобы быстро вылечиться.

нажмите правую кнопку, чтобы взять гранату (если она доступна). нажмите правую кнопку еще раз, чтобы бросить гранату (нажимайте, чтобы бросить ее дальше).
нажмите левую кнопку, чтобы открыть меню.

навигация - 2/вверх и 8/вниз. нажмите 5/ок, чтобы подтвердить выбор.
