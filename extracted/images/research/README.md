# Graphics research artifacts

Эта папка предназначена для ручных исследовательских артефактов (annotated PNG, side-by-side сравнения, заметки по конкретным кадрам), которые используются вместе с гипотезами из:

- `docs/reverse_engineering/graphics_hypotheses.md`

Рекомендуемая структура:

- `extracted/images/research/<pack>/chunk_<NN>/...`
- рядом хранить `notes.md` с ссылками на `frame_XXX.png` / `frame_XXX.bin`.
